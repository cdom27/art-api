**Artist API** is a free and lightweight Web API about the 50 most influential artists of all time.

You can try it out by [generating a public api key](https://art.cidominguez.com/docs) (don't worry it's free).

This repository manages the source code and public domain content including photos of artworks provided in v1 of this api.

## Tech

v1 is deployed on **Google Cloud Platform**.

- Express.js (Node.js)
- TypeScript
- PostgreSQL
- Drizzle
- Bcrypt
- Zod
