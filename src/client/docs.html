Introduction

The Influential Artist API provide data for the 50 most inlfuential artists of all time. As well as images and data of their artwork's within public domain.

Getting Started

Generate a key

Register your domain to generate a Public API Key


        


        
        

          
Set Environment Variables


          

            Create a `.env` file in the root of your project and add the
            following environment variables:
          


          
Public Key


          
Public key generated for your domain.


           IAA_API_KEY= 

          
Set Base URL


           IAA_BASE_URL=https://art.cidominguez.com/api/v1
